# Online Exam System

A modern full-stack online examination platform built with Node.js, Express.js, MongoDB, Next.js, and Tailwind CSS.

## Features

- User Authentication (JWT)
- Role-based Access (Admin / Student)
- Online Exams
- Automatic Timer
- Auto Save Answers
- Resume Exam After Disconnection
- Exam Results
- Admin Dashboard
- Responsive Design

## Tech Stack

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT

### Frontend

- Next.js
- React
- Tailwind CSS
- Axios
- React Query

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
